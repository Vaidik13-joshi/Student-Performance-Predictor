# Student-Performance-Predictor
Machine learning model that predicts student results using Logistic Regression with features like study hours and attendance.
## 📌 Overview

Predicts whether a student will pass or fail using a Machine Learning model based on:

* Study Hours
* Attendance

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn

## 🚀 How to Run

pip install -r requirements.txt
python main.py

## 📊 Example

Input:

* Study Hours: 6
* Attendance: 72

Output:

* Prediction: Pass

## ⚠️ Note

This project uses a small dataset for demonstration purposes.
